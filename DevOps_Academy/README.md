# PostgreSQL Fundamentals - DevOps Academy

Welcome to the PostgreSQL Fundamentals course! This academy is designed for complete beginners who want to learn PostgreSQL from scratch.

## Course Overview

This course consists of 10 hands-on exercises that will teach you the fundamentals of PostgreSQL. By the end of this course, you will be able to:

- Set up PostgreSQL using Docker
- Create and manage databases and tables
- Insert, query, update, and delete data
- Understand and use indexes for performance
- Manage users and permissions
- Perform database maintenance tasks
- Backup and restore databases

## Prerequisites

- Basic command line knowledge
- Docker installed on your system
- A text editor of your choice

## Exercises

1. **Exercise 1: Setting Up PostgreSQL with Docker** - Learn how to run PostgreSQL in a Docker container
2. **Exercise 2: Creating Your First Database** - Create and connect to a PostgreSQL database
3. **Exercise 3: Creating Tables** - Learn about data types and create your first table
4. **Exercise 4: Inserting and Querying Data** - Add data to tables and retrieve it using SELECT
5. **Exercise 5: Understanding Indexes** - Create indexes and see how they improve query performance
6. **Exercise 6: Updating and Deleting Data** - Modify and remove data from tables
7. **Exercise 7: Dropping Tables** - Learn how to safely remove tables and understand dependencies
8. **Exercise 8: Creating Users and Managing Permissions** - Set up database users and control access
9. **Exercise 9: VACUUM Operations** - Understand database maintenance and the VACUUM command
10. **Exercise 10: Backup and Restore** - Learn how to backup and restore your databases

## How to Use This Course

1. Start with Exercise 1 and work through them sequentially
2. Each exercise includes:
   - Learning objectives
   - Step-by-step instructions
   - Practice tasks
   - Key concepts explained
   - Common pitfalls to avoid

3. Take your time with each exercise - understanding the fundamentals is more important than speed
4. Try the practice tasks at the end of each exercise to reinforce your learning

## Getting Help

If you get stuck:
- Review the exercise instructions carefully
- Check the PostgreSQL documentation: https://www.postgresql.org/docs/
- Try breaking down the problem into smaller steps
